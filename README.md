# FuelFlex Platform

FuelFlex Platform est une plateforme SaaS multi-tenant destinée à la gestion professionnelle des stations-service.

## Composants

- Backend : Java 21 et Spring Boot
- Frontend Web : React ou Next.js
- Application Mobile : Flutter
- Base de données : PostgreSQL
- Infrastructure : Docker et Docker Compose
- Gestion de versions : Git et GitHub

## Structure

- `backend/` : API Spring Boot
- `frontend/` : application web
- `mobile/` : application mobile destinée aux pompistes
- `infrastructure/` : Docker, Nginx et scripts
- `database/` : migrations et données initiales
- `documentation/` : documents techniques et fonctionnels
